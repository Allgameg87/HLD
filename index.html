SmartHome Hub - High Level Design

Выполнил: Чиркин Елизар
Дата: 7 апреля 2025 г.

1. Введение

1.1. Назначение документа

Документ описывает высокоуровневую архитектуру системы управления умным домом "SmartHome Hub", включая компоненты, взаимодействия и основные технологии.

1.2. Цель системы

Создание централизованной платформы для управления IoT-устройствами в доме (освещение, климат, безопасность, мультимедиа) с поддержкой голосового управления, сценариев автоматизации и удаленного доступа.

2. Архитектура системы

2.1. Общая схема

Система строится по микросервисной архитектуре с использованием MQTT для обмена сообщениями между устройствами.

Компоненты системы:

Frontend

  • Мобильное приложение (iOS/Android, React Native)
  • Веб-интерфейс (React.js)
  • Голосовой ассистент (интеграция с Alexa/Google Assistant)

Backend

  • API Gateway (Node.js) – единая точка входа для запросов
  • Auth Service – аутентификация и авторизация (OAuth 2.0, JWT)
  • Device Manager – регистрация и управление устройствами
  • Automation Engine – выполнение сценариев

Базы данных

  • PostgreSQL – хранение пользователей, устройств, логов
  • Redis – кэширование и временные данные

IoT-устройства

  • Датчики (температура, движение) → ESP8266/Arduino
  • Исполнительные устройства (реле, умные розетки)

Интеграции

  • Умные колонки (Alexa Skills Kit)
  • Погодные сервисы (OpenWeatherMap)

3. Технологии

Node.js NestJS React React Native PostgreSQL Redis MQTT WebSockets Docker Kubernetes AWS/GCP

4. Безопасность

  • Шифрование TLS для MQTT/HTTP
  • Ролевая модель доступа (гость, пользователь, администратор)
  • Регулярные аудиты уязвимостей

5. Дополнительные функции

AI-аналитика

Предсказание расходов на электричество на основе исторических данных и прогноза погоды.

Режим "Эко"

Автоматическое отключение неиспользуемых устройств для экономии энергии.

Интеграция с календарем

Сценарии на основе событий ("Если в календаре 'отпуск' → включить режим охраны").
